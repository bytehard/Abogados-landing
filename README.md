# Abogados Landing

Landing page estatica para una campana de orientacion juridica familiar gratuita en Bolivia.

## Estructura

- `index.html`: pagina principal.
- `css/`: estilos de Bootstrap, Font Awesome y estilos propios.
- `js/`: scripts de interaccion.
- `images/`: imagenes y fondos de la landing.

## Uso local

Abra `index.html` en el navegador para revisar la pagina.
