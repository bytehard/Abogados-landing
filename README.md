# Abogados Landing

Landing page estática para una campaña de orientación jurídica familiar gratuita en Bolivia.

## Estructura

- `index.html`: página principal.
- `css/`: estilos de Bootstrap, Font Awesome y estilos propios.
- `js/`: scripts de interacción.
- `images/`: imágenes y fondos de la landing.

## Uso local

Abra `index.html` en el navegador para revisar la página.

## Pendientes de configuración

- Reemplazar el bloque de video por el video real de presentación.
- Configurar el número de WhatsApp en `js/custom.js` o conectar el formulario a una herramienta de agenda.
